# 🐾 ねこゴロゴロ癒し

10匹の可愛い猫たちと、あなただけの愛猫を癒し音にまとめた個人アプリ。

---

## 🚀 アプリの開き方

### Macの場合

1. **Finder（フォルダの画面）** を開く
2. `デスクトップ` → `yuyu_Desk` → `yuyu_Claude` → `neko-gorogoro` の順に開く
3. **`index.html` をダブルクリック**
4. Safari や Chrome が起動してアプリが開きます🎉
5. 初回起動時、自動的に **10匹のデフォルト猫** が登録されます🐈🐱🐈‍⬛

### スマホで使う場合

スマホで使うには、いくつか方法があります（やりたくなったら一緒にやりましょう）：
- **方法A**: パソコンとスマホを同じWi-Fiにつなぎ、簡易サーバーで開く
- **方法B**: GitHub Pages などで無料公開する
- **方法C**: スマホの中にファイルを送って開く

## 🎮 使い方

### 🐱 猫を追加する

1. 画面下の **「＋ 猫を追加」** ボタンを押す
2. 名前・写真（任意）・音声ファイルを選ぶ
3. **「保存する」** で完了！

### ▶️ 再生・停止する

- 猫の右側の **▶️ ボタン** で再生
- もう一度押すと停止

### ⏱️ タイマー機能

- 上のタイマーで「5分・10分・30分・60分」を選ぶと、自動で停止します
- 「なし」を選ぶとずっと再生されます

### 🔊 音量調整

- スライダーで音量を変更
- 再生中にもリアルタイムで変わります

### ⭐ お気に入り

- 各猫の **☆マーク** を押すと、お気に入りに登録
- 上の **「⭐ お気に入り」** タブで、お気に入りだけ表示できます

### 💌 共有（友達に送る）

- 各猫の **💌マーク** を押す
- **「💾 音声ファイルをダウンロード」** を押すとファイルが保存される
- そのファイルを **LINE / AirDrop / メール** などで友達に送れます
- 受け取った友達は、自分のアプリで「猫を追加」から読み込めば、同じ音を楽しめます🎵

### ✏️ 編集・🗑️ 削除

- **✏️マーク**: 名前や写真、音声を変更できます
- **🗑️マーク**: 削除します（確認ダイアログが出ます）

## 💾 データはどこに保存されているの？

- **猫の情報・音声・写真**: ブラウザ内の安全な保管場所（IndexedDB）に保存
- **音量・タイマー設定**: ブラウザ内のメモ帳（localStorage）に保存

すべて**あなたのパソコン・スマホの中だけ**に保存されます。外部に送信されません🔒

### ⚠️ データが消えるケース

- ブラウザの「履歴・キャッシュをすべて削除」を実行した時
- 別のブラウザに変えた時（例: SafariからChromeへ）
- シークレットモード（プライベートブラウジング）で開いた時

大事な猫の音は、定期的に💌マークから**バックアップ用にダウンロード**しておくと安心です。

## ❓ うまく動かない時

| 症状 | 確認すること |
|---|---|
| 音が出ない | パソコンの音量、ブラウザの音量、アプリの音量スライダー |
| ファイルが選べない | mp3、m4a、wavなど一般的な音声ファイルか確認 |
| 猫が表示されない | ページを更新（Cmd+R）してみる |
| アプリが開かない | 別のブラウザ（Safari/Chrome）で試してみる |
